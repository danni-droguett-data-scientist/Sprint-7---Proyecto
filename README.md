# Sprint-7---Proyecto
El objetivo de este proyecto es proporcionarte más posibilidades de practicar las tareas habituales de la ingeniería de software. Gracias a estas tareas, podrás aumentar y complementar tus habilidades en el campo de los datos y te convertirás en un candidato más atractivo para posibles empleadores.

## Descripción del proyecto

Este proyecto tiene como objetivo practicar tareas habituales de la ingeniería de software aplicadas al campo de los datos. A través de su desarrollo, se busca reforzar habilidades técnicas importantes como la creación y gestión de entornos virtuales de Python, la organización de un proyecto y el desarrollo de una aplicación web.

El proyecto utiliza un conjunto de datos de anuncios de venta de coches como base para construir una aplicación interactiva. Sin embargo, el enfoque principal no está en realizar un análisis exhaustivo del dataset, sino en aplicar buenas prácticas de desarrollo, estructurar correctamente el entorno de trabajo y crear una herramienta funcional que permita visualizar o explorar la información de manera sencilla.

Esta práctica permite complementar las habilidades de análisis de datos con competencias de desarrollo de software, aumentando así la preparación para proyectos reales y mejorando el perfil profesional frente a posibles empleadores.


## Funcionalidad de la aplicación web

La aplicación web fue desarrollada con Streamlit y permite explorar de forma interactiva el conjunto de datos de anuncios de venta de vehículos en Estados Unidos.

Actualmente, la aplicación incluye las siguientes funcionalidades:

- Visualización de un histograma de la columna `odometer`, que permite observar la distribución del kilometraje de los vehículos publicados.
- Visualización de un gráfico de dispersión entre `odometer` y `price`, que permite analizar la relación entre el kilometraje y el precio de los vehículos.

La aplicación utiliza pandas para cargar los datos desde el archivo `vehicles_us.csv` y plotly para construir gráficos interactivos.
